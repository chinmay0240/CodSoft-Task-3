# Iris Flower Classification 

## What This Project Does
This project predicts the species of an iris flower using machine learning.  
The model takes flower measurements like petal length, petal width, sepal length, and sepal width, and predicts whether the flower is:
- Setosa
- Versicolor
- Virginica
  
## What We Have Done In This Project
- Loaded the Iris dataset using Scikit-learn
- Converted flower labels into species names
- Split the dataset into training and testing data
- Used StandardScaler for data scaling
- Built a Support Vector Machine (SVM) model
- Trained the model using machine learning
- Tested the model accuracy
- Generated classification report and confusion matrix
- Predicted flower species for sample input data
- Visualized the dataset using Matplotlib
